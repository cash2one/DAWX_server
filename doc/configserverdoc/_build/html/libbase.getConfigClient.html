libbase.getConfigClient module¶

getConfigClient¶

用来连接 Sockertserver 包括（ CSserver 和 NDserver ）。主体函数 getres

libbase.getConfigClient.getres(astr, host='', port=1011)[source]¶

向 socketserver发送一个 astr 。 server采用密码验证，所以需要线进行密码判断，然后再传送 * astr * 字符串。 收到服务器传送的数据之后，进行长度验证，如果不正确，重新传送。 判断成功或者失败次数超出限制之后端开连接，返回结果。

Parameters:
  • astr – 字符串命令
  • host – 发送命令的server ip。 默认是给CSserver发送，
  • port – server port。 默认是CSserver 的port。
